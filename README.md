# Go Practice 

## Things to Practice on:
> Problems categorized by concepts generated using claude.ai 

---

**Arrays & Slices**
1. [x] Reverse an array in-place
2. [x] Find the largest and smallest element in an array
3. [ ] Remove duplicates from a sorted array
4. [ ] Given an array, return all pairs that sum to a target number
5. [ ] Rotate an array by k steps

---

**Strings**
1. [ ] Check if a string is a palindrome
2. [ ] Count the frequency of each character
3. [ ] Check if two strings are anagrams
4. [ ] Find the longest common prefix in a list of strings
5. [ ] Implement your own `contains(s, substr string) bool`

---

**Linked Lists** *(implement your own, don't use a library)*
1. [ ] Implement a singly linked list with insert, delete, print
2. [ ] Reverse a linked list
3. [ ] Detect a cycle in a linked list
4. [ ] Find the middle node of a linked list
5. [ ] Merge two sorted linked lists

---

**Stacks & Queues** *(implement using slices)*
1. [ ] Implement a stack with push, pop, peek
2. [ ] Check if parentheses in a string are balanced — `{[()]}` valid, `{[(])}` not
3. [ ] Implement a queue using two stacks
4. [ ] Implement a min-stack that returns the minimum element in O(1)

---

**Hash Maps**
1. [ ] Find the first non-repeating character in a string
2. [ ] Group anagrams together from a list of strings
3. [ ] Given an array, find if any two numbers add up to a target (without nested loops)
4. [ ] Find the longest consecutive sequence in an unsorted array

---

**Recursion**
1. [ ] Fibonacci (then optimize it)
2. [ ] Factorial
3. [ ] Binary search recursively
4. [ ] Flatten a nested slice `[]interface{}`
5. [ ] Tower of Hanoi

---

**Sorting**
1. [ ] Implement bubble sort
2. [ ] Implement merge sort
3. [ ] Implement quicksort
4. [ ] Given a sorted rotated array, find a target element

---

**Trees** *(implement your own binary tree)*
1. [ ] Insert and search in a BST
2. [ ] Print a tree level by level (BFS)
3. [ ] Find the height of a tree
4. [ ] Check if a binary tree is symmetric
5. [ ] Find the lowest common ancestor of two nodes

---

**Graphs** *(represent as adjacency list using a map)*
1. [ ] Implement BFS and DFS
2. [ ] Detect a cycle in an undirected graph
3. [ ] Find the shortest path between two nodes
4. [ ] Check if a graph is bipartite
